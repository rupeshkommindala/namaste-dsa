# LinkedList is a linear data structure
# Nodes are linked together through a reference field
# Two types of LinkedList
# Singly LinkedList
    # Pointer points to the next node
    # Node = Data + pointer
# Doubly LinkedList
    # Pointer points to the next node and to its previous node
    # Node = Data + 2 pointer
# Head is the starting node of the LinkedList
# LinkedList is represented by its Head
# Tail points to the null

# LinkedList
    # Linear
    # Non-contiguous
    # Dynamic-size(change easily)
    # Node = value+pointer
    # Fetching Element is hard - O(n)
    # Insertion/Deletion is easily - O(n)
    # Extra Memory

# Arrays
    # Linear
    # Contiguous
    # Fixed-size(Can be Dynamic)
    # Just value
    # Fetching element is easily - O(1)
    # Insertion/Deletion is complex - O(n)
    # Memory Efficient
